# Obsidian Folder Structure

此 Obsidian 仓库的文件存储路径如下：

```sh
Vault/Theme/Project(Type)/Note.md

Theme:
    - Curio
    - Knowledge
    - Misc
    - Museion
    - Tutorials
```

对于较复杂的 `Project`，将 `Note.md` 作为索引，同级创建 `detail` 文件夹存储项目的所有文件。
